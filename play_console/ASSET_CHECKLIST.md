# Play Console Asset Checklist

Use this checklist before submitting a production release.

## Required App Details

- App name: `Taqyim DZ`
- Package name: `com.intilak.taqyimdz`
- Developer name: `Seddik Bouyakoub`
- Company: `Intilak`
- Contact email: `appdevybs@gmail.com`

## Graphics

- App icon: 512 x 512 PNG (32-bit), max 1 MB
- Feature graphic: 1024 x 500 PNG or JPG, max 15 MB
- Phone screenshots: minimum 2
  - Recommended: 1080 x 1920 (or similar 16:9)
- 7-inch tablet screenshots: optional but recommended
- 10-inch tablet screenshots: optional but recommended

## Content and Policy

- Short description (max 80 chars)
- Full description
- Data Safety form completed
- App access instructions (if any)
- Ads declaration
- Content rating questionnaire
- Target audience and age declaration
- Privacy policy URL (recommended / required depending on data usage)

## Release Files

- Upload AAB: `build/app/outputs/bundle/release/app-release.aab`
- Upload key fingerprint: `play_console/upload_key_fingerprint.txt`
- Release notes: `play_console/release_notes_en-US.txt`

## Final Verification

- Install release build on device and open app
- Test workbook import, edit, validation, export
- Confirm language switching works (AR/FR/EN)
- Confirm no crashes during export and reopen flows
